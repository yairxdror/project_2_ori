About:

Hello! I'm Yair, 27 years old, a student with a passion for technology, web development, and user experience design.


About the site:

This site was built as a personal project to allow users to track digital currencies in real time. The site includes features such as:

  • A dummy report on the market value of digital currencies.
  • Ability to mark selected currencies and create a custom report.
  • View selected currencies on a personal chart.
  • A modern and interactive user experience.
  • Support for quick search by currency name.
  • Smart caching of information to improve user experience (cache).

Website Goals:

My main goal was to create a friendly and simple platform that would help users understand the world of digital currencies in an accessible way. I believe that everyone deserves the opportunity to learn and get to know this field, and the website is the first step towards that.
